# Status Definitions

| Status | Description |
|----------|-------------|
| Draft | Under development |
| Review | Under validation |
| Active | In production |
| Deprecated | Scheduled for retirement |
| Archived | No longer used |

---

# Criticality Levels

| Level | Description |
|--------|-------------|
| Low | Minimal impact |
| Medium | Operational impact |
| High | Business impact |
| Critical | Enterprise impact |

---

# Required Metadata

Every workflow must contain:

- Workflow ID
- Name
- Description
- Owner
- Environment
- Dependencies
- Version
- Status
- Last Update

---

# Lifecycle

Draft → Review → Active → Deprecated → Archived
